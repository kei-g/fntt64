# fntt64 [![license][license-image]][license-url]

`fntt64` - Fast Number Theoretic Transform 64x64

## Abstract

$$
7^{2^{26}\left(2^{32}-1\right)}\equiv{2^{39}}\mod{2^{64}-2^{32}+1}
$$
